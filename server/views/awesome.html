How to use imagify

Resize image
params Description
token Signup to receive token - required
url Your image URL( only jpg, png and gif formats) - required
resize_width New width for image - optional if resize_height is provided
resize_height New height for image - optional if resize_width is provided
Examples

  1. resize image to 300px width
    imagify image: http://imagify.co/img? token=047fg33&url=http://bit.ly/1Mjm2rb&resize_width=300

  2. resize image to 500px height
    imagify image: http://imagify.co/img? token=047fg33&url=http://bit.ly/1Mjm2rb&resize_height=500

  3. resize image to 300px width & 500px height
    imagify image: http://imagify.co/img? token=047fg33&url=http://bit.ly/1Mjm2rb&resize_width=300&resize_height=500

Note: when you only provide one dimension, e.g 1 & 2 above, the other dimension size will be auto based on the aspect ratio of original image size

Crop a rectangle from center
params Description
token Signup to receive token - required
url Your image URL ( only jpg, png and gif formats) - required
crop_width Width of the rectangle to crop from the center of the image. - required
crop_height Height of the rectangle to crop from the center of the image. - required
Example

  1. Crop an image from center, 500px width by 200px height
    imagify image: http://imagify.co/img? token=047fg33&url=http://bit.ly/1Mjm2rb&crop_width=500&crop_height=200
